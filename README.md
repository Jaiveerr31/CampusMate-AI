# 🎓 CampusMate AI

CampusMate AI is a modern academic management web application designed to help students organize their studies efficiently.

## ✨ Features

- 📊 Dashboard
- 🤖 AI Assistant
- 📝 Assignment Manager
- 📅 Timetable
- 📚 Notes Manager
- 📈 CGPA Calculator
- ✅ Attendance Tracker
- ⚙️ Settings
- 🔐 Secure Login Interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage
