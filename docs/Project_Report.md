# CampusMate AI

## Project Overview

CampusMate AI is a student productivity platform developed to simplify academic management.

## Objectives

- Improve student productivity
- Organize academic tasks
- Provide an AI-powered study assistant
- Track academic performance

## Modules

- Landing Page
- Login
- Dashboard
- AI Assistant
- Assignments
- Attendance
- CGPA Calculator
- Notes
- Timetable
- Settings

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage

## Future Scope

- Backend Integration
- Authentication System
- Cloud Database
- AI API Integration
- Mobile Application
